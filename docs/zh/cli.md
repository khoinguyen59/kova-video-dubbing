# KOVA CLI compatibility note

This repository is branded and documented as **KOVA**. The main application
surfaces are Vietnamese and English. This file replaces a legacy imported CLI
document so the repository no longer presents an unrelated product or Chinese
interface as a KOVA feature.

## Supported command-line role

The CLI is a compatibility surface for scripted local jobs. The product flow is
the desktop application, which uses five explicit user-approved stages:

1. Video source
2. Translation and subtitles
3. Fixed dubbing voice
4. Video output and tuning
5. Outputs

Do not use a CLI command to bypass the review gates in the desktop project
timeline. For the current configuration, security requirements, and source
attribution, read [KOVA documentation](../KOVA_DOCUMENTATION.md) and the
implementation report in the repository root.
